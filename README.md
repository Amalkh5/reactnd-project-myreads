# MyReads Project

This is the starter template for the final assessment project for Udacity's React Fundamentals course.


## Project Setup

1. Clone the Project - `git clone https://github.com/Amalkh5/reactnd-project-myreads.git`
2. Go into the directory where the project now lives - `cd reactnd-project-myreads`
3. Install the dependencies - `npm install`
4. Start the app - `npm start`
